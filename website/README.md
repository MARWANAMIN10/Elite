# Elite Real Estate Website

A responsive real estate website with property listings, user profiles, and contact functionality.

## Live Demo
[Your GitHub Pages URL will appear here after deployment]

## Features
- User authentication (login/register) with localStorage
- Property listings with detailed views
- User profile management
- Contact agents and company information
- Responsive design for all devices

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage for session management
- Base64 encoded images for offline demonstration

## Pages
1. **index.html** - Login/Registration page
2. **home.html** - Landing page with featured properties
3. **properties.html** - Complete property listings
4. **details.html** - Detailed property view
5. **profile.html** - User profile and statistics
6. **contact.html** - Contact form and company information

## Setup Instructions
1. Clone the repository: `git clone https://github.com/YOUR_USERNAME/elite-real-estate.git`
2. Open `index.html` in your web browser
3. Register a new account or use any credentials to login

## How to Use
1. Register a new account on the login page
2. Browse properties on the home or properties page
3. Click "View Details" to see more information about a property
4. Contact agents directly from property details
5. View your profile to see account information
